# Keyboard Shortcuts

## Global

| Action | Windows / Linux | macOS |
|---|---|---|
| Global search | `Ctrl+K` | `Cmd+K` |
| New deck | `Ctrl+N` | `Cmd+N` |
| Save | `Ctrl+S` | `Cmd+S` |

## Deck Workshop ‚Äî General

| Action | Windows / Linux | macOS |
|---|---|---|
| Focus card search | `Ctrl+F` | `Cmd+F` |
| Export deck | `Ctrl+E` | `Cmd+E` |
| Toggle sidebar | `Ctrl+B` | `Cmd+B` |
| Toggle AI assistant | `Ctrl+I` | `Cmd+I` |
| Undo | `Ctrl+Z` | `Cmd+Z` |
| Redo | `Ctrl+Shift+Z` | `Cmd+Shift+Z` |

## Canvas ‚Äî Navigation

| Action | Windows / Linux | macOS |
|---|---|---|
| Pan | Hold `Space` + drag | Hold `Space` + drag |
| Zoom in | `+` or scroll up | `+` or scroll up |
| Zoom out | `-` or scroll down | `-` or scroll down |
| Reset zoom (100%) | `Ctrl+0` | `Cmd+0` |
| Fit all to viewport | `Ctrl+Shift+0` | `Cmd+Shift+0` |

## Canvas ‚Äî Mode

| Action | Key |
|---|---|
| Toggle Select / Hand mode | `H` |

## Canvas ‚Äî Adding items

| Action | Key |
|---|---|
| Add group | `G` |
| Add sticky note | `N` |
| Add widget | `W` |

## Canvas ‚Äî Selection

| Action | How |
|---|---|
| Select item | Click |
| Add to selection | `Shift` + click |
| Rubber-band select | Click and drag on empty canvas (Select mode) |
| Select all | `Ctrl+A` / `Cmd+A` |
| Deselect all | `Escape` |

## Canvas ‚Äî Editing

| Action | Windows / Linux | macOS |
|---|---|---|
| Delete selected items | `Delete` or `Backspace` | `Delete` or `Backspace` |
| Duplicate selected | `Ctrl+D` | `Cmd+D` |

## Deck Workshop ‚Äî Tabs

| Action | Key |
|---|---|
| Workshop (canvas) tab | `1` |
| List tab | `2` |
| Missing tab | `3` |
| Combos tab | `4` |
