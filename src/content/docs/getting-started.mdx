# Getting Started

## Installation

### Windows

1. Download the latest `.exe` installer from the [Releases page](https://github.com/karn-mtg/forge/releases)
2. Run the installer — no admin rights required
3. Launch **Karn Forge** from the Start menu or desktop shortcut

### macOS

1. Download the `.dmg` file from the [Releases page](https://github.com/karn-mtg/forge/releases)
2. Open the `.dmg` and drag Karn Forge to your Applications folder
3. On first launch, right-click → Open to bypass Gatekeeper

## First Launch

On first launch, Karn Forge downloads the Scryfall card database (~50 MB). This takes about 30 seconds on a typical connection. After that, everything works fully offline.

## Creating Your First Deck

1. Click **New Deck** in the left sidebar
2. Give your deck a name and choose a format
3. Use the **Card Search** panel on the right to find cards
4. Drag cards onto the canvas or click the **+** button to add them
5. Press `Ctrl+S` / `Cmd+S` to save

## Importing a Deck

1. Open **File → Import** from the menu bar
2. Choose your source:
   - **Paste text** — any standard decklist format (one card per line)
   - **Moxfield URL** — paste a public deck URL
   - **Archidekt URL** — paste a public deck URL
   - **CSV file** — import from a spreadsheet
3. Karn Forge fetches card data automatically

## Keyboard Shortcuts

| Action | Windows | macOS |
|---|---|---|
| New Deck | `Ctrl+N` | `Cmd+N` |
| Save | `Ctrl+S` | `Cmd+S` |
| Search Cards | `Ctrl+F` | `Cmd+F` |
| Export Deck | `Ctrl+E` | `Cmd+E` |
| Toggle Sidebar | `Ctrl+B` | `Cmd+B` |
