# Installation

Karn Forge is distributed as a native desktop installer for Windows, macOS, and Linux.

## Download

Go to the [Releases page](https://github.com/karn-mtg/forge/releases) and download the installer for your platform:

| Platform | File |
|---|---|
| Windows | `.exe` installer |
| macOS | `.dmg` disk image |
| Linux | `.AppImage` |

<Tabs>
  <TabItem label="Windows">
    <Steps>
      1. Download the `.exe` installer from the Releases page.
      2. Run the installer. No administrator rights are required.
      3. Launch **Karn Forge** from the Start menu or the desktop shortcut.
    </Steps>
  </TabItem>
  <TabItem label="macOS">
    <Steps>
      1. Download the `.dmg` file from the Releases page.
      2. Open the `.dmg` and drag Karn Forge to your Applications folder.
      3. On first launch, right-click the app icon → **Open** to bypass Gatekeeper.
    </Steps>

    <Aside type="note">
      macOS may show a security warning because the app is not notarized. Right-clicking and choosing **Open** bypasses this warning and only needs to be done once.
    </Aside>
  </TabItem>
  <TabItem label="Linux">
    <Steps>
      1. Download the `.AppImage` file from the Releases page.
      2. Mark it as executable: `chmod +x KarnForge-*.AppImage`
      3. Run it: `./KarnForge-*.AppImage`
    </Steps>

    <Aside type="note">
      Some Linux distributions require `libfuse2` to run AppImages. Install it with your package manager if the app does not launch (e.g., `sudo apt install libfuse2` on Ubuntu).
    </Aside>
  </TabItem>
</Tabs>

## First launch

On first launch, Karn Forge opens the **Arsenal** setup wizard. Arsenal is the local data service that manages the card database and rules database.

<Steps>
  1. Click **Set Up Arsenal** in the prompt that appears.
  2. Wait for the three components to download: the server binary, the cards database (~50 MB), and the rules database.
  3. Once setup is complete, the app reloads and is ready to use.
</Steps>

<Aside type="tip">
  After the initial setup, everything works fully offline. You can refresh the databases later from the Arsenal panel whenever updates are available.
</Aside>

## System requirements

| | Minimum |
|---|---|
| OS | Windows 10, macOS 12, Ubuntu 20.04 |
| RAM | 4 GB |
| Disk | 500 MB free (for card database and app data) |
| Network | Required only for first setup and optional database updates |

## Updating

Karn Forge checks for app updates automatically a few seconds after launch. When a new version is available, a notification appears in **Settings → Updates** with a download button. Once the download completes, click **Install** to restart and apply the update.

The card database and rules database can be updated independently from the **Arsenal** panel without reinstalling the app.
