# Bem-vindo ao Karn Forge

Karn Forge é uma aplicação desktop gratuita e de código aberto para construir e gerenciar decks de Magic: The Gathering.

## O que é o Karn Forge?

O Karn Forge combina um poderoso construtor visual de decks com gerenciamento de coleção, análises detalhadas e dados de cartas em tempo real — tudo em um aplicativo desktop rápido e que funciona offline.

<CardGrid>
  <Card title="Canvas Visual" icon="pencil">
    Organize cartas em um canvas infinito. Agrupe por estratégia, adicione notas e visualize seu deck como um designer.
  </Card>
  <Card title="Rastreamento de Coleção" icon="document">
    Importe do Moxfield, Archidekt ou CSV. Acompanhe preços em tempo real.
  </Card>
  <Card title="Análises" icon="information">
    Curva de mana, distribuição de cores, divisão por tipo — otimize com dados.
  </Card>
  <Card title="Dados ao Vivo" icon="star">
    Powered by Scryfall. Cada carta, cada edição, sempre atualizado.
  </Card>
</CardGrid>
