# Primeiros Passos

## Instalação

### Windows

1. Baixe o último instalador `.exe` na [página de Releases](https://github.com/karn-mtg/forge/releases)
2. Execute o instalador — sem necessidade de direitos de administrador
3. Abra o **Karn Forge** pelo Menu Iniciar ou atalho na área de trabalho

### macOS

1. Baixe o arquivo `.dmg` na [página de Releases](https://github.com/karn-mtg/forge/releases)
2. Abra o `.dmg` e arraste o Karn Forge para a pasta Aplicativos
3. Na primeira abertura, clique com o botão direito → Abrir para ignorar o Gatekeeper

## Primeira Inicialização

Na primeira inicialização, o Karn Forge baixa o banco de dados de cartas do Scryfall (~50 MB). Isso leva cerca de 30 segundos em uma conexão típica. Depois disso, tudo funciona totalmente offline.

## Criando Seu Primeiro Deck

1. Clique em **Novo Deck** na barra lateral esquerda
2. Dê um nome ao deck e escolha um formato
3. Use o painel de **Busca de Cartas** à direita para encontrar cartas
4. Arraste as cartas para o canvas ou clique no botão **+** para adicioná-las
5. Pressione `Ctrl+S` / `Cmd+S` para salvar

## Importando um Deck

1. Abra **Arquivo → Importar** na barra de menu
2. Escolha a origem:
   - **Colar texto** — qualquer formato de lista padrão (uma carta por linha)
   - **URL do Moxfield** — cole a URL de um deck público
   - **URL do Archidekt** — cole a URL de um deck público
   - **Arquivo CSV** — importe de uma planilha
3. O Karn Forge busca os dados das cartas automaticamente

## Atalhos de Teclado

| Ação | Windows | macOS |
|---|---|---|
| Novo Deck | `Ctrl+N` | `Cmd+N` |
| Salvar | `Ctrl+S` | `Cmd+S` |
| Buscar Cartas | `Ctrl+F` | `Cmd+F` |
| Exportar Deck | `Ctrl+E` | `Cmd+E` |
| Alternar Sidebar | `Ctrl+B` | `Cmd+B` |
