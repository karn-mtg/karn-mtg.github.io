# Willkommen bei Karn Forge

Karn Forge ist eine kostenlose, quelloffene Desktop-Anwendung zum Erstellen und Verwalten von Magic: The Gathering Decks.

## Was ist Karn Forge?

Karn Forge kombiniert einen leistungsstarken visuellen Deck-Builder mit Sammlungsverwaltung, detaillierten Analysen und Live-Kartendaten — alles in einer schnellen, offline-fähigen Desktop-App.

<CardGrid>
  <Card title="Visuelles Canvas" icon="pencil">
    Karten auf einem unendlichen Canvas anordnen. Nach Strategie gruppieren, Haftnotizen hinzufügen und das Deck wie ein Designer visualisieren.
  </Card>
  <Card title="Sammlungs-Tracker" icon="document">
    Import von Moxfield, Archidekt oder CSV. Preise in Echtzeit verfolgen.
  </Card>
  <Card title="Analysen" icon="information">
    Manakurve, Farbverteilung, Typ-Aufschlüsselung — mit Daten optimieren.
  </Card>
  <Card title="Live-Daten" icon="star">
    Powered by Scryfall. Jede Karte, jeder Druck, immer aktuell.
  </Card>
</CardGrid>
