# Tastaturkürzel

## Global

| Aktion | Windows / Linux | macOS |
|---|---|---|
| Globale Suche | `Ctrl+K` | `Cmd+K` |
| Neues Deck | `Ctrl+N` | `Cmd+N` |
| Speichern | `Ctrl+S` | `Cmd+S` |

## Deck Workshop — Allgemein

| Aktion | Windows / Linux | macOS |
|---|---|---|
| Kartensuche fokussieren | `Ctrl+F` | `Cmd+F` |
| Deck exportieren | `Ctrl+E` | `Cmd+E` |
| Seitenleiste umschalten | `Ctrl+B` | `Cmd+B` |
| KI-Assistenten umschalten | `Ctrl+I` | `Cmd+I` |
| Rückgängig | `Ctrl+Z` | `Cmd+Z` |
| Wiederholen | `Ctrl+Shift+Z` | `Cmd+Shift+Z` |

## Canvas — Navigation

| Aktion | Windows / Linux | macOS |
|---|---|---|
| Schwenken | `Space` gedrückt halten + ziehen | `Space` gedrückt halten + ziehen |
| Hineinzoomen | `+` oder Mausrad nach oben | `+` oder Mausrad nach oben |
| Herauszoomen | `-` oder Mausrad nach unten | `-` oder Mausrad nach unten |
| Zoom zurücksetzen (100%) | `Ctrl+0` | `Cmd+0` |
| Alles in den Viewport einpassen | `Ctrl+Shift+0` | `Cmd+Shift+0` |

## Canvas — Modus

| Aktion | Taste |
|---|---|
| Auswahl- / Handmodus umschalten | `H` |

## Canvas — Elemente hinzufügen

| Aktion | Taste |
|---|---|
| Gruppe hinzufügen | `G` |
| Haftnotiz hinzufügen | `N` |
| Widget hinzufügen | `W` |

## Canvas — Auswahl

| Aktion | Wie |
|---|---|
| Element auswählen | Klick |
| Zur Auswahl hinzufügen | `Shift` + Klick |
| Auswahlrahmen | Klick und Ziehen auf leerem Canvas (Auswahlmodus) |
| Alle auswählen | `Ctrl+A` / `Cmd+A` |
| Auswahl aufheben | `Escape` |

## Canvas — Bearbeiten

| Aktion | Windows / Linux | macOS |
|---|---|---|
| Ausgewählte Elemente löschen | `Delete` oder `Backspace` | `Delete` oder `Backspace` |
| Auswahl duplizieren | `Ctrl+D` | `Cmd+D` |

## Deck Workshop — Tabs

| Aktion | Taste |
|---|---|
| Workshop (Canvas)-Tab | `1` |
| Listen-Tab | `2` |
| Fehlend-Tab | `3` |
| Combos-Tab | `4` |
