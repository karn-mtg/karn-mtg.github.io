# Erste Schritte

## Installation

### Windows

1. Lade den neuesten `.exe`-Installer von der [Releases-Seite](https://github.com/karn-mtg/forge/releases) herunter
2. Führe den Installer aus — keine Administratorrechte erforderlich
3. Starte **Karn Forge** über das Startmenü oder die Desktop-Verknüpfung

### macOS

1. Lade die `.dmg`-Datei von der [Releases-Seite](https://github.com/karn-mtg/forge/releases) herunter
2. Öffne die `.dmg`-Datei und ziehe Karn Forge in deinen Programme-Ordner
3. Beim ersten Start: Rechtsklick → Öffnen, um Gatekeeper zu umgehen

## Erster Start

Beim ersten Start lädt Karn Forge die Scryfall-Kartendatenbank herunter (~50 MB). Das dauert bei einer typischen Verbindung etwa 30 Sekunden. Danach funktioniert alles vollständig offline.

## Dein erstes Deck erstellen

1. Klicke auf **Neues Deck** in der linken Seitenleiste
2. Gib deinem Deck einen Namen und wähle ein Format
3. Nutze das **Kartensuche**-Panel auf der rechten Seite, um Karten zu finden
4. Ziehe Karten auf das Canvas oder klicke auf die **+**-Schaltfläche
5. Drücke `Strg+S` / `Cmd+S` zum Speichern

## Ein Deck importieren

1. Öffne **Datei → Importieren** aus der Menüleiste
2. Wähle deine Quelle:
   - **Text einfügen** — jedes Standard-Decklisten-Format (eine Karte pro Zeile)
   - **Moxfield-URL** — füge eine öffentliche Deck-URL ein
   - **Archidekt-URL** — füge eine öffentliche Deck-URL ein
   - **CSV-Datei** — aus einer Tabellenkalkulation importieren
3. Karn Forge ruft automatisch Kartendaten ab

## Tastenkürzel

| Aktion | Windows | macOS |
|---|---|---|
| Neues Deck | `Strg+N` | `Cmd+N` |
| Speichern | `Strg+S` | `Cmd+S` |
| Karten suchen | `Strg+F` | `Cmd+F` |
| Deck exportieren | `Strg+E` | `Cmd+E` |
| Seitenleiste umschalten | `Strg+B` | `Cmd+B` |
