# Welcome to Karn Forge

Karn Forge is a free, open-source desktop application for building and managing Magic: The Gathering decks.

## What is Karn Forge?

Karn Forge combines a powerful visual deck builder with collection management, deep analytics, and live card data — all in a fast, offline-capable desktop app.

<CardGrid>
  <Card title="Visual Canvas" icon="pencil">
    Arrange cards on an infinite canvas. Group by strategy, add sticky notes, and visualize your deck like a designer.
  </Card>
  <Card title="Collection Tracker" icon="document">
    Import from Moxfield, Archidekt, or CSV. Track prices in real time.
  </Card>
  <Card title="Analytics" icon="information">
    Mana curve, color distribution, type breakdown — optimize with data.
  </Card>
  <Card title="Live Data" icon="star">
    Powered by Scryfall. Every card, every printing, always up-to-date.
  </Card>
</CardGrid>
