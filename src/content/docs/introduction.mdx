# Welcome to Karn Forge

Karn Forge is a free, open-source desktop application for building, managing, and analyzing Magic: The Gathering decks. It combines a visual canvas builder with collection tracking, live card data, and an AI assistant — all in a fast, offline-capable app.

## What you can do with Karn Forge

<CardGrid>
  <Card title="Dashboard" icon="rocket">
    Get an instant overview of your library: activity charts, color distribution, format breakdown, and your most-used cards across all decks.
  </Card>
  <Card title="Deck Workshop" icon="pencil">
    Build decks on an infinite visual canvas. Arrange cards into groups, add sticky notes, place analytics widgets, and save multiple named arrangements of the same deck.
  </Card>
  <Card title="Card Search" icon="magnifier">
    Search the full Scryfall card database. Filter by name, type, color, format legality, and more. Click any result to see full card details and add it directly to your deck.
  </Card>
  <Card title="Collection" icon="document">
    Track your physical card collection with condition grades (NM, LP, MP, HP, DMG), foil flags, and per-card prices. Assign cards to recipients to manage trades and orders.
  </Card>
  <Card title="Wishlist" icon="star">
    Keep a prioritized list of cards you want to acquire. Set priorities from Low to Critical, reorder by drag, and add cards directly to any of your decks.
  </Card>
  <Card title="AI Assistant" icon="comment">
    Ask an AI assistant for deck advice, card suggestions, and strategy help. The assistant can show card previews, offer choice menus, and maintain a conversation history per deck.
  </Card>
  <Card title="Arsenal" icon="setting">
    The Arsenal is Karn Forge's local data service. It manages the Scryfall card database and comprehensive rules database so the app works fully offline after the first setup.
  </Card>
  <Card title="Widgets" icon="bars">
    Embed live analytics directly on the deck canvas. Built-in widgets include Mana Curve, Color Distribution, Type Breakdown, Deck Stats, Land Ratio, Draw Odds, and EDHREC Ranks — or build your own with JavaScript.
  </Card>
</CardGrid>

## Supported formats

Karn Forge supports deck building and validation for all major MTG formats:

- **Commander** (EDH)
- **Modern**
- **Standard**
- **Pioneer**
- **Legacy**
- **Vintage**
- **Pauper**

## Offline-first

After the first launch, Karn Forge downloads the Scryfall card database via the Arsenal service. All card lookups, search, and deck validation then work fully offline. The database is updated independently and can be refreshed whenever you like from the Arsenal panel.
