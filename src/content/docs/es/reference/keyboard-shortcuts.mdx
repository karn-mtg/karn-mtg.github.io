# Atajos de teclado

## Globales

| Acción | Windows / Linux | macOS |
|---|---|---|
| Búsqueda global | `Ctrl+K` | `Cmd+K` |
| Nuevo mazo | `Ctrl+N` | `Cmd+N` |
| Guardar | `Ctrl+S` | `Cmd+S` |

## Deck Workshop — General

| Acción | Windows / Linux | macOS |
|---|---|---|
| Enfocar búsqueda de cartas | `Ctrl+F` | `Cmd+F` |
| Exportar mazo | `Ctrl+E` | `Cmd+E` |
| Alternar barra lateral | `Ctrl+B` | `Cmd+B` |
| Alternar asistente de IA | `Ctrl+I` | `Cmd+I` |
| Deshacer | `Ctrl+Z` | `Cmd+Z` |
| Rehacer | `Ctrl+Shift+Z` | `Cmd+Shift+Z` |

## Canvas — Navegación

| Acción | Windows / Linux | macOS |
|---|---|---|
| Desplazar | Mantén `Space` + arrastra | Mantén `Space` + arrastra |
| Acercar | `+` o desplaza hacia arriba | `+` o desplaza hacia arriba |
| Alejar | `-` o desplaza hacia abajo | `-` o desplaza hacia abajo |
| Restablecer zoom (100%) | `Ctrl+0` | `Cmd+0` |
| Ajustar todo al viewport | `Ctrl+Shift+0` | `Cmd+Shift+0` |

## Canvas — Modo

| Acción | Tecla |
|---|---|
| Alternar modo Selección / Mano | `H` |

## Canvas — Añadir elementos

| Acción | Tecla |
|---|---|
| Añadir grupo | `G` |
| Añadir nota adhesiva | `N` |
| Añadir widget | `W` |

## Canvas — Selección

| Acción | Cómo |
|---|---|
| Seleccionar elemento | Clic |
| Añadir a la selección | `Shift` + clic |
| Selección por área | Clic y arrastra en canvas vacío (modo Selección) |
| Seleccionar todo | `Ctrl+A` / `Cmd+A` |
| Deseleccionar todo | `Escape` |

## Canvas — Edición

| Acción | Windows / Linux | macOS |
|---|---|---|
| Eliminar elementos seleccionados | `Delete` o `Backspace` | `Delete` o `Backspace` |
| Duplicar selección | `Ctrl+D` | `Cmd+D` |

## Deck Workshop — Pestañas

| Acción | Tecla |
|---|---|
| Pestaña Workshop (canvas) | `1` |
| Pestaña Lista | `2` |
| Pestaña Faltantes | `3` |
| Pestaña Combos | `4` |
