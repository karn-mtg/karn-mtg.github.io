# Primeros Pasos

## Instalación

### Windows

1. Descarga el instalador `.exe` más reciente desde la [página de Releases](https://github.com/karn-mtg/forge/releases)
2. Ejecuta el instalador — no se requieren derechos de administrador
3. Lanza **Karn Forge** desde el menú de inicio o el acceso directo del escritorio

### macOS

1. Descarga el archivo `.dmg` desde la [página de Releases](https://github.com/karn-mtg/forge/releases)
2. Abre el `.dmg` y arrastra Karn Forge a tu carpeta de Aplicaciones
3. En el primer inicio, clic derecho → Abrir para evitar Gatekeeper

## Primer inicio

En el primer inicio, Karn Forge descarga la base de datos de cartas de Scryfall (~50 MB). Esto tarda unos 30 segundos con una conexión típica. Después, todo funciona completamente sin conexión.

## Creando tu primer mazo

1. Haz clic en **Nuevo Mazo** en la barra lateral izquierda
2. Dale un nombre a tu mazo y elige un formato
3. Usa el panel de **Búsqueda de Cartas** en la derecha para encontrar cartas
4. Arrastra cartas al canvas o haz clic en el botón **+** para añadirlas
5. Pulsa `Ctrl+S` / `Cmd+S` para guardar

## Importar un mazo

1. Abre **Archivo → Importar** desde la barra de menú
2. Elige tu fuente:
   - **Pegar texto** — cualquier formato de lista de mazos estándar (una carta por línea)
   - **URL de Moxfield** — pega una URL de mazo público
   - **URL de Archidekt** — pega una URL de mazo público
   - **Archivo CSV** — importa desde una hoja de cálculo
3. Karn Forge obtiene los datos de las cartas automáticamente

## Atajos de teclado

| Acción | Windows | macOS |
|---|---|---|
| Nuevo Mazo | `Ctrl+N` | `Cmd+N` |
| Guardar | `Ctrl+S` | `Cmd+S` |
| Buscar Cartas | `Ctrl+F` | `Cmd+F` |
| Exportar Mazo | `Ctrl+E` | `Cmd+E` |
| Alternar Barra lateral | `Ctrl+B` | `Cmd+B` |
