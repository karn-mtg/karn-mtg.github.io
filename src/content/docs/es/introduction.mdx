# Bienvenido a Karn Forge

Karn Forge es una aplicación de escritorio gratuita y de código abierto para construir, gestionar y analizar mazos de Magic: The Gathering. Combina un constructor visual con canvas, seguimiento de colección, datos de cartas en tiempo real y un asistente de IA — todo en una aplicación rápida con capacidad offline.

## Qué puedes hacer con Karn Forge

<CardGrid>
  <Card title="Dashboard" icon="rocket">
    Obtén una visión instantánea de tu biblioteca: gráficos de actividad, distribución de colores, desglose por formato y tus cartas más usadas en todos tus mazos.
  </Card>
  <Card title="Deck Workshop" icon="pencil">
    Construye mazos en un canvas visual infinito. Organiza cartas en grupos, añade notas adhesivas, coloca widgets de análisis y guarda múltiples disposiciones con nombre para el mismo mazo.
  </Card>
  <Card title="Búsqueda de cartas" icon="magnifier">
    Busca en la base de datos completa de Scryfall. Filtra por nombre, tipo, color, legalidad de formato y más. Haz clic en cualquier resultado para ver los detalles completos y añadir la carta directamente a tu mazo.
  </Card>
  <Card title="Colección" icon="document">
    Rastrea tu colección física de cartas con grados de condición (NM, LP, MP, HP, DMG), indicadores de foil y precios por carta. Asigna cartas a destinatarios para gestionar intercambios y pedidos.
  </Card>
  <Card title="Lista de deseos" icon="star">
    Mantén una lista priorizada de cartas que quieres adquirir. Define prioridades de Baja a Crítica, reordena arrastrando y añade cartas directamente a cualquiera de tus mazos.
  </Card>
  <Card title="Asistente de IA" icon="comment">
    Consulta a un asistente de IA para obtener consejos sobre mazos, sugerencias de cartas y ayuda con estrategias. El asistente puede mostrar vistas previas de cartas, ofrecer menús de opciones y mantener un historial de conversación por mazo.
  </Card>
  <Card title="Arsenal" icon="setting">
    El Arsenal es el servicio de datos local de Karn Forge. Gestiona la base de datos de cartas de Scryfall y la base de datos de reglas completa para que la app funcione completamente offline tras la configuración inicial.
  </Card>
  <Card title="Widgets" icon="bars">
    Integra análisis en vivo directamente en el canvas del mazo. Los widgets integrados incluyen Curva de Maná, Distribución de Colores, Desglose por Tipo, Estadísticas del Mazo, Ratio de Tierras, Probabilidades de Robo y Rankings de EDHREC — o crea los tuyos con JavaScript.
  </Card>
</CardGrid>

## Formatos soportados

Karn Forge admite la construcción de mazos y validación para todos los formatos principales de MTG:

- **Commander** (EDH)
- **Modern**
- **Standard**
- **Pioneer**
- **Legacy**
- **Vintage**
- **Pauper**

## Primero offline

Tras el primer inicio, Karn Forge descarga la base de datos de cartas de Scryfall a través del servicio Arsenal. A partir de entonces, todas las búsquedas de cartas, consultas y validaciones de mazos funcionan completamente offline. La base de datos se actualiza de forma independiente y puede refrescarse cuando quieras desde el panel de Arsenal.
