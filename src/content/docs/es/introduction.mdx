# Bienvenido a Karn Forge

Karn Forge es una aplicación de escritorio gratuita y de código abierto para construir y gestionar mazos de Magic: The Gathering.

## ¿Qué es Karn Forge?

Karn Forge combina un potente constructor visual de mazos con gestión de colección, análisis detallado y datos de cartas en vivo — todo en una aplicación de escritorio rápida con capacidad offline.

<CardGrid>
  <Card title="Canvas Visual" icon="pencil">
    Organiza cartas en un canvas infinito. Agrupa por estrategia, añade notas adhesivas y visualiza tu mazo como un diseñador.
  </Card>
  <Card title="Rastreador de colección" icon="document">
    Importa desde Moxfield, Archidekt o CSV. Rastrea precios en tiempo real.
  </Card>
  <Card title="Análisis" icon="information">
    Curva de maná, distribución de colores, desglose de tipos — optimiza con datos.
  </Card>
  <Card title="Datos en vivo" icon="star">
    Powered by Scryfall. Cada carta, cada edición, siempre actualizado.
  </Card>
</CardGrid>
